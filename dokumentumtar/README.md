# Dokumentumtár (MA/MSc hallgatói anyagok)

Ez a mappa a mesterszakos hallgatóknak szóló **letölthető dokumentumokat**
tartalmazza (mintatantervek, záróvizsga-tételek, diplomamunka-sablonok,
etikai és plágium-nyilatkozatok, felvételi tájékoztatók).

Ezeket a régi tanszéki oldalról hoztuk át. A korábbi helyük:

- `master_CogSci/` ← régen: `http://www.cogsci.bme.hu/~ktkuser/master_CogSci/`
  (Számítógépes és kognitív idegtudomány MSc)
- `master_pszi/` ← régen: `http://www.cogsci.bme.hu/~ktkuser/master_pszi/`
  (Pszichológia MA)

## FONTOS: a belső linkek megőrzése

A mappaszerkezetet **szándékosan változatlanul** tartottuk meg, hogy a
fájlokra mutató belső hivatkozások továbbra is működjenek. Az új címek a
régi útvonalat tükrözik, csak az elejük változott:

```
régi:  http://www.cogsci.bme.hu/~ktkuser/master_CogSci/Zarovizsga/<fájl>
új:    https://ktt.ttk.bme.hu/dokumentumtar/master_CogSci/Zarovizsga/<fájl>
```

Ezért **a fájlokat ne nevezd át és ne helyezd át**, csak ha a rájuk mutató
linkeket is frissíted. Új dokumentumot a megfelelő almappába tölts fel.

A felhasználóknak szánt, linkelt lista a `/dokumentumtar/` oldalon jelenik
meg (forrása a repó gyökerében a `dokumentumtar.md`). Az a lista
automatikusan felsorolja az itt lévő fájlokat, így új fájl feltöltése után
külön szerkesztés nélkül megjelenik.
